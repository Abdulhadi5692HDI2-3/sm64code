How to use your code:

After you hit the button to generate the code for the version of the game you're running, you'll have to add the code into your emulator as a new cheat code. In Project64, this can be done through the top menu bar at System>>Cheats. Click the arrow on the right hand side to open the cheat creation section. Paste your code into the box labeled "address," and give a name for the code in the "Name" box. Click on "Add Cheat," and it will appear in the list on the left hand side. Now just check off the box beside it and Mario will be sporting his new look!
